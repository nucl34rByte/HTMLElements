Texts

To print a simple text without any tags, just type it in HTML. For example:

This is a simple example

And if you want to use a paragraph, just type this text between two p tags. For example:

<p>This is a paragraph.</p>

Headers

To use a header, just print a text between two h1 or h2 or h3 or h4 or h5 or h6 tags. For example:

<h3>This is a header.</h2>

Fonts 3

Times New Roman Calibri Arial Black
Fonts

To use fonts, just print a text between two font tags with a "face" attribute, which contains the font's name. For example:

<font face="Times New Roman">This is a Times New Roman font.</font>

Text's styles4

Bold Italic Deleted Underlined
This's bold text. This's italic text. This's deleted text. This's underlined text.
Text's styles

To contol a style of a text, just print a text between two b(bold text) or i(italic text) or del(deleted text) or u(underlined text) tags. For example:

<b>This's bold text.</b> <i>This's italic text.</i> <del>This's deleted text.</del> <u>This's underlined text.</u>

Text's alignes

To align a text, just print a text between two p or span tags with an attribute "align", which contains an aligned type (center, right, left, justify). For example:

<p align="left">This's aligned text.</p>

Preformated texts 1

This text is preformated!
Preformated texts

To use a preformated text (with spaces, new lines,...), just print a text between two pre tags. For example:

<pre>This's preformated text.</pre>

Text's sizes 3

This is a big text. This is a normal text. This is a small text.
Text's sizes

To control a text's sizes, just print a text between two font tags with an attribute "size", which contains a text's size. For example:

<font size="10">This's a big text.</font>

Text's colors 5

Red Green Blue Black
Text's colors

To change a text's color, just type the text between two font tags with an attribute "color", which contains a hexademical RGB color or color's name. For example:

<font color="#ff0000">Red</font> <font color="red">Red</font>

Images

To use images on the page, use img tag with a "src" attribute, which contains a path to the image. For example:

<img src="res/image.png"/>

Lists

To use lists on the page, use ul(unordered list) or ol(ordered list) tags with a li(list item) tag in it's body. For example:

<ul><li>1</li><li>2</li><li>3</li></ul>

Definitions

To use a definitions on the page use dl(definition list) with a dt(definition term) and dd(definition description) tags in it's body. For example:

<dl> <dt>1 term</dt> <dd>1 description</dd> <dt>2 term</dt> <dd>2 description</dd> </dl>

Forms

To use a forms on the page use a form tag with a input tag in it's body. For example:

<form> Your name: <input type="text"/> </form>
